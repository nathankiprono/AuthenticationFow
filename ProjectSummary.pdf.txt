AuthenticationFlow – Project Summary

Project Title:
AuthenticationFlow – Modern ASP.NET Core Authentication with Email OTP & JWT

Tech Stack:
- ASP.NET Core MVC (C#)
- Entity Framework Core
- SQL Server
- JWT (JSON Web Token)
- Bootstrap 5 & Bootstrap Icons
- jQuery
- SMTP (Gmail) for email

Folder Structure:
- Controllers/ – API and web controllers
- DTOs/ – Data Transfer Objects
- Models/ – Entity models
- Services/ – Business logic, OTP/email
- Data/ – EF Core DbContext, migrations
- Views/ – Razor UI (SignUp, SignIn, VerifyOtp)
- wwwroot/ – Static assets
- appsettings.json – Config (DB, JWT, SMTP)

Key Features:
- User registration with email OTP verification
- Secure login with JWT token
- Only verified users can log in
- Modern, responsive UI (Bootstrap, icons, loader)
- Email sending via Gmail SMTP

Setup & Run:
1. Clone repo: git clone <your-repo-link>
2. Update appsettings.json with your DB, JWT, SMTP
3. Run migrations: dotnet ef database update
4. Start: dotnet run or via Visual Studio
5. Visit: https://localhost:<port>

Challenges Faced:
- Email deliverability (Gmail SMTP, app passwords)
- OTP security (demo: DB, production: expiry/rate limit)
- UI/UX polish
- JWT integration
